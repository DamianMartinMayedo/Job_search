# Job Search CRM

## Product Overview
Personal CRM for managing job search applications. Track companies, contacts, messages, and job offers in one place.

## Target Users
Job seekers managing multiple applications across different companies.

## Core Features
- Company management with contacts and notes
- Email message tracking with drafts and follow-ups
- Job offer aggregation and application tracking
- Email templates for outreach
- Document management (CVs, cover letters)

## Design Principles
- Clean, professional interface for daily use
- Fast interactions with minimal friction
- Clear information hierarchy
- Responsive design for desktop and mobile

## Tech Stack
- React 19 + Vite
- TailwindCSS 4
- TanStack Query for data fetching
- PostgreSQL backend via Netlify Functions

## Current State
Functional MVP with room for polish and refinement.
